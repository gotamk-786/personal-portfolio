# Portfolio Website — Gotam Kumar

Modern, responsive portfolio for Gotam Kumar (Computer Science student and Software & Systems Developer). Built with static HTML/CSS/JS—no build tools required.

## Run locally
1. Clone/download the repo.
2. Open `index.html` in your browser (double‑click or use a simple HTTP server).

## Structure
- `index.html` — content and sections (hero, about, skills, projects, education, services, contact).
- `styles.css` — core styling and layout.
- `enhancements.css` — animations, gradients, glass effects, premium touches.
- `script.js` — navigation toggle, smooth scroll, scroll animations, skill bar animation, scroll spy, quick contact modal, counters, filters.
- `profile/` — profile image(s).

## Resume download
The “Download Résumé” button expects `assets/resume.pdf`. Add your PDF there or adjust the link in `index.html`.

## Customization tips
- Update text/content in `index.html`.
- Swap profile image in `profile/`.
- Tweak colors/spacing in `styles.css` and animations in `enhancements.css` (keep theme consistent).
- If you add new sections, mirror existing class patterns for styles/animations.

## Accessibility & UX
- Keyboard-friendly nav toggle.
- Smooth scrolling with scroll padding for fixed navbar.
- Visible focus/hover states on buttons/links/cards.
